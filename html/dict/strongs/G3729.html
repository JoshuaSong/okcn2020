ὁρμάω
to set in motion, urge
ὁρμάω to start, spur or urge on, i.e. (reflexively) to dash or plunge
Derivation: from G3730;
KJV Usage: run (violently), rush. G3730
Thayer:
1) to set in rapid motion, stir up, incite, urge on
2) to start forward impetuously, to rush
