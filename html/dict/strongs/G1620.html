ἐκτίθημι
to set out, place outside
ἐκτίθημι to expose; figuratively, to declare
Derivation: from G1537 and G5087;
KJV Usage: cast out, expound. G1537 G5087
Thayer:
1) to place or set out, put outside, expose
2) to set up, exhibit
3) metaph. to set forth, declare, expound
