גּרל גּורל
gôrâl gôrâl
go-rawl', go-ral'
From an unused root meaning to be rough (as stone); properly a pebble, that is, a lot (small stones being used for that purpose); figuratively a portion or destiny (as if determined by lot)
KJV Usage: lot.

Brown-Driver-Briggs' Hebrew Definitions

גּרל גּורל

1. lot
a. lot - pebbles used for systematically making decisions
2. portion
a. lot, portion (thing assigned by casting lots)
b. recompense, retribution
Origin: from an unused root meaning to be rough (as stone)
