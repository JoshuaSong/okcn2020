ἐπιστέλλω
to send to, send as a message
ἐπιστέλλω to enjoin (by writing), i.e. (genitive case) to communicate by letter (for any purpose)
Derivation: from G1909 and G4724;
KJV Usage: write (a letter, unto). G1909 G4724
Thayer:
1) to send one a message, command
2) to write a letter
3) to enjoin by letter, to write instructions
